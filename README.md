# BattleKit

BattleKit is your MVP scaffold. This repo is set up for a Next.js + TypeScript app with Jest tests, GitHub Actions CI, and Vercel deploy previews. It is designed to be agent-friendly so coding agents can implement features via Pull Requests you can review safely.

## Stack
- Next.js (TypeScript)
- Node 20
- Jest + Testing Library
- GitHub Actions (CI)
- Vercel (preview deploys + production)

## Run locally
```bash
npm ci
npm run dev  # http://localhost:3000
```

## Tests
```bash
npm test
```

## Build
```bash
npm run build
```
